# Term project : Micromouse  
### [Micromouse 소개]  
#### 자율주행 자동차 구현
  -	전방과 좌/우측에 센서를 달아 장애물을 감지하고 회피하는 동작
  -	신호등의 신호를 수신하여 자동차 출발/정지 제어
  -	사용자에게 자동차의 상태를 LCD로 출력
